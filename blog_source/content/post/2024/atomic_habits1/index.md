《掌控习惯：如何养成好习惯并戒除坏习惯》（英文名：*Atomic Habits*）是 2024 年读的第一本书。这本书的英文版总结，之前在社交媒体上看到过好几次，但是一直没在意，直到有一天看到有人推荐这本书的中文版，仔细看才知道原来它就是 *Atomic Habits*，所以 2024 年开年便开始读这本书。在阅读本书的过程中，我发现这本书不只是简单教你如何养成习惯，更从生物进化的角度让你理解人的天性，如何更好的利用天性来养成习惯。所以在阅读时就决定，在读完后一定要好好总结一下，本文便是对《掌控习惯》的总结，分享给大家。另外，如果只看一本习惯养成（自律）类的书，强烈推荐本书！

# 正确认识习惯

想象这样一种场景：新年伊始，我们信心满满地列出一长串计划，满怀期待的迎接新的一年。也许我们决定健身，也许是打算学习一门新的语言，也许是计划精打细算以积攒一部分存款。

但是当我们开始按照自己的规划努力（自律）一段时间后，发现自己并没有什么明显的变化，于是我们可能怀疑自己是否能做到，在这种自我怀疑中，慢慢的放弃。

当年底的钟声敲响时，我们发现年初定的计划都没有实际的进展。这时失落和自责充满了内心，我们可能觉得自己的自制力太差了，控制不住自己，于是对自己发誓，新的一年，一定一定要更加努力。然而，仿佛历史总在重演，每到年底，我们总陷于原地踏步的自责。久而久之，我们被这种似乎无法打破的循环困扰，越来越陷入自我怀疑。于是，我们可能不再相信自己有能力去实现那些年初设定的目标。是我们真的不够优秀，不够自律？还是我们的方法本身就有问题？

我们对习惯总是存在错误的认知：
1. 总是关注短期变化；
2. 总是以目标为导向。

然而，事实并非如此。
## 积跬步，以至千里

第一个错误认知：总是关注短期变化，忽略长期可能带来的影响。

我们总是高估某个决定性时刻的重要性，也很容易低估每天进行微小改进的价值。我们常常说服自己，大规模的成功需要大规模的行动。我们会给自己施加压力，让自己努力作出一些人人都想谈论的惊天动地的改进，但事实并不是这样。

**习惯是自我提高的复利**，就像金钱借助复利倍增一样，我们的习惯的效果也会随着不断重复而倍增。

{{< figure src="img/习惯的复利.jpg" >}}

所以，**此时此刻是成就辉煌还是一事无成并不重要，重要的是我们当前的习惯是否让我们走向了通往成功的道路。我们应该重点关注自己当下前进的轨迹，而不是已经取得了什么样的结果**。

我们很难养成持久习惯的核心原因之一是：**最有力的结果总是姗姗来迟**。

我们做了一些小小的改变，过一段时间没有看到任何效果，于是放弃了。例如，我们想减肥，于是决定坚持跑步，结果坚持了一个月，发现体重没有任何变化，内心不由出现跑步没有用的想法：“我都坚持了一个月了，怎么身体没有任何变化？”一旦这种想法占了上风，就会轻而易举的放弃。但是事实上，这里的关键点在于：**要想实现有意义的改变，养成新的习惯，需要坚持足够长的时间，这样才能突破没有明显变化的平台期，或者更好的说法是潜能积蓄期**。

我们总是期望进步是线性的，或者说至少期望它有立竿见影的效果，但是事实上的进步却是一条曲线，如下图所示：

## 忘记目标，专注于体系（过程）

第二个错误认知：总是以目标为导向，忽略体系（过程）的重要性。

我们要区分目标和体系，目标是我们想要达到的结果，而体系则是涉及这些结果的过程。例如：

| 角色   | 目标                           | 体系                                                                     |
| ------ | ------------------------------ | ------------------------------------------------------------------------ |
| 教练   | 让自己带的队伍赢得冠军         | 招募球员、管理助理教练和训练的方式                                       |
| 企业家 | 创建一家营业额上百万美元的企业 | 测试产品创意、雇用员工和开展营销活动                                     |
| 音乐家 | 演奏一支新曲子                 | 你练习的频率、你如何分解和处理高难度曲段，以及你从导师那里获得反馈的方法 |

任何一项运动的终极目标都是取得最好成绩，但是在整场比赛中都死盯着计分牌则是毫无意义的。争取每天训练取得进步是获得最好成绩的唯一方法。设定目标的目的是赢得比赛，构建体系的目的是持续参与这项赛事。

所以，**目标确定大方向，体系促进我们进步！** 关注目标，同时更应该注重过程——过程即奖励！

# 习惯塑造身份，身份让习惯保持

改变我们的行为，养成好的习惯具有挑战性，原因在于：
1. 我们没有找对试图改变的东西；
2. 我们试图以错误的方式改变自己的行为习惯。

## 行为改变的三个层次

事实上，对于行为的改变，我们可以将其划分成三个层次：
1. **第一层是改变你的结果**：这一层次事关改变结果，设定的目标大多数与这个层次的改变相关；
2. **第二层是改变你的过程**：这一层次涉及改变习惯和体系，例如定时去锻炼等；
3. **第三层是改变你的身份**：这一层次是有关改变的信念，即我们的世界观、自我形象以及对自己和他人所做的判断。我们的大多数信念、建设和偏见与这个层次有关。

{{< figure src="img/行为改变的三个层次.jpg" >}}

很多人开始改变自己的习惯时，把注意力集中在自己想要表达的目标上。**但正确的做法是：培养基于自己身份的习惯。借助这种方式，我们的关注点会落在我们希望自己成为什么样的人**。说想要那样的人是一回事，而说我本身就是那种人则是另一回事。越是以自己身份的某一方面为傲，越有动力保持与之相关的习惯。这也许就是自我暗示的强大力量。

{{< figure src="img/基于结果和身份的习惯.jpg" >}}

## 改变身份的两个步骤

改变身份是一个简单的两步过程：
1. 决定成为哪类人；
2. 用小赢证明给自己看。

{{< figure src="img/改变身份的两步.png" >}}

身份决定过程，过程中的小赢又会强化这种身份认知，从形成正向循环，直到有一天，你会发现自己真的成为了那样的人。

身份改变是习惯改变的北极星，想要改变行为，也许我们真正要问自己的是：“你是否想要成为那样的人？”，否则，我们对改变的追求就像一艘没有舵的船，始终无法走在正确的航向上。

# 培养良好习惯的正确方式

在介绍培养良好习惯的正确方式之前，我们先看看大脑为什么要培养习惯。

## 大脑为何会培养习惯

>- 习惯是重复了足够多次数后变得自动化的行为
>- 习惯形成的过程始于反复尝试
>- 习惯减轻了认知负荷，释放了心智能力，从而让你可以将注意力分配给其他任务

每当我们遇到新的情况，大脑就要做出决定：“我该如何应对？” 在这个过程中，大脑的神经高度活跃，会仔细分析当前的形势，并有意识地决定如何行动。我们在此过程中接受了海量信息，并试图从中理出头绪，而大脑此时也正忙于学习最有效的行动路径。当我们采取大脑给出的行动计划，并获得一个意想不到的奖励后，我们便会对今后的策略做出调整。

这便是人类行动背后的反馈回路：尝试、失败、学习，然后进行不同的尝试。经过一番练习，那些无用的动作逐渐消失，而有用的动作得到加强。这就是正在形成的习惯。每当我们反复面对一个问题，大脑就会把解决这个问题的过程自动化，而我们说的**习惯就是解决我们经常面临的问题和压力的一系列自动化解决方案**。

随着习惯的形成，大脑的活跃程度逐渐降低，当在未来类似的情况出现时，我们的大脑会直接跳过错误环节，并创立一条心理规则：如果是这种情况，就用那种应对方式。

习惯的形成非常有用，因为心理意识是大脑的瓶颈。一心不能二用。因此，大脑总是努力确保你集中注意力，关注当下最根本的问题。**只要有可能，头脑会有意识地把一些任务交给无意识去自动完成。这正是习惯形成时会发生的情况。习惯减轻了认知负荷，释放了心智能力，从而让你可以将注意力分配给其他任务**。

## 养成习惯的四个步骤

基于以上分析，我们可以得出养成习惯的过程可以分为四个简单步骤：揭示、渴求、反应和奖励。

{{< figure src="img/习惯的形成.jpg" >}}

这四步模式是每个习惯的核心支柱，我们的大脑每次都以同样的顺序运行这些步骤。

### 提示

提示触发行为，是预测奖励的信息：史前祖先寻找重要奖励线索，如食物、水和异性。现代人关注次要奖励线索，如金钱、名誉、权力、地位、赞扬、认可、爱情、友谊和个人满足感。大脑不断分析环境，寻找奖励线索。线索引发渴求，是接近奖励的第一个迹象。

### 渴求

渴求是习惯背后的动力，驱使我们采取行动： **我们渴求的是习惯所带来的状态变化，而不是习惯本身**。渴求与改变内在状态的强烈愿望相关，这一点非常重要。渴求因人而异，不同的提示会引发不同的欲望。观察者的思想、感觉和情感将提示转化为渴求。

### 反应

反应是实际习惯的表现，可以是思想或行动上的：**反应取决于动力大小和行动的难易程度。如果行动过于艰难，你不会采取反应。反应也受到能力的限制。习惯只会出现在你具备相应能力的情况下**。如果你想扣篮但跳不高，无法达到篮筐，那就别再想了，以后也不必考虑这件事了。

### 奖励

获得奖励是养成每个习惯的最终目标：提示的作用让我们注意到奖励的存在。渴求是想要得到奖励，反应则是获得奖励。我们追求奖励是因为其有两个作用：
1. 满足我们的需求
2. 对我们有益

奖励满足渴求是其主要作用。奖励本身具有益处，如食物和水提供生存所需能量，晋升获取金钱和尊重，保持身材改善健康，成功的约会等。奖励能迅速满足我们对吃、地位或认可的渴求，带来暂时的解脱和满足感。

其次，奖励教会我们记住哪些行为并应用于未来。**大脑是奖励探测器，感觉神经系统不断监测能满足欲望和带来快乐的行为**。快乐和失望是反馈机制的一部分，帮助大脑区分有用和无用的行为。奖励完成习惯形成循环，终结反馈回路。

{{< figure src="img/习惯循环.jpg" >}}

总的来说：
- 提示，触发渴求；
- 渴求，激发反应；
- 反应，提供满足渴求的奖励，并最终与提示相关联。

这四个步骤一起形成了一个神经反馈回路——提示、渴求、反应、奖励，提示、渴求、反应、奖励——并最终让你养成自然而然的习惯，由此构成完整的习惯循环。

# 行为转变四大定律

基于以上习惯发挥作用的原理，我们可以总结出行为转变的四大定律：
- 第一定律（提示）：让它显而易见
- 第二定律（渴求）：让它有吸引力
- 第三定律（反应）：让它简单易行
- 第四定律（奖励）：让它令人愉悦

所以每当我们想要改变自己的行为时，可以问自己几个简单问题：
1. 我怎样才能让它显而易见？
2. 我怎样才能让它有吸引力？
3. 我怎样才能让它简便易行？
4. 我们怎样能让它令人愉悦？
